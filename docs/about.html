About

Escape! is a text based adventure game where you play the main role! Your goal is to get out of the mysterious dungeon that you somehow ended up in. How you do this is entirely up to you though...

The game has several different endings. Can you get them all?


Available Languages

  • English
  • Swedish / Svenska
  • German / Deutsch

Do you want to contribute by translating the game to another language? Feel free to open a pull request on the Github repository with your translations!

The German translations might be a little questionable due to the fact that I, the developer, am not proficient enough in the language to translate the entire game without the help of translation software.

Frequently asked questions

  • Which platforms is the game available on?

    • The .exe file can only run on Windows as far as I know, but the entire code is available in the Github repository, so feel free to download it and run it in a code editor, or convert the .exe or the .py file to a format that can run on your OS.

  • How many endings are there?

    • The game has five endings.

  • Can I contribute to the game's content?

    • No, pull requests that feature new game content will be denied. You can however submit new translations through pull requests. If you have feedback on the game, or feature suggestions you can open a pull request and upload a .txt file.

  • What age rating is Escape!?

    • Escape! is made to fit all ages.
