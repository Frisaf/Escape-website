What is Escape!?

Escape is a text adventure game where you choose your own destiny! There are several different endings and many rooms to explore. There may or may not be some secret easter eggs somewhere too...

If you like chill text based adventure games with fantasy themes and puzzles, the mysterious escape room Escape! will surely fit you!

How much does the game cost?

The game is free! You don't need to pay a single penny to play it.
